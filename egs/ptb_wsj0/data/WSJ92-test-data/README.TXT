1000best.sent
the 1000-bset list for performing rescoring, which is generated by Kaldi based a DNN-based acoustic model.

1000bset.acscore
the acoustic score for the 1000-best list

1000best.lmscore
the lmscore score of the original ngram LM, which can be replaced to the results of new LMs.

orcale.wer
record the orcale WER of the 1000-best list.
